# `-=SnjApp=-`

### React Native Cross-platform Application

## Instalation:

    sudo npm i -g expo-cli
    expo init SnjHotApp

## Development enviropment:

    cd SnjHotApp
    npm install
    npm start

## Build:

    sudo npm install -g eas-cli
    eas build -p android
    expo build:status

## Publish

    expo publish
    eas update
